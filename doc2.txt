    1. Login
登录界面


Function Description功能说明：Login user name and password to enter the AS/RS Warehouse Executing System management system登录用户名称与密码进入AS/RS Warehouse Executing System管理系统。
    1. Enter the user name and password, select the default language输入用户名称和密码，选择默认语言。
    2. Click "Login", the user enters the WES system单击‘Login’，用户进入WES系统。
    3. Click ‘Exit’ to exit the login interface单击‘Exit’，退出登录界面。




    2. Main Operation Menu主操作菜單


The main screen is divided into five areas主屏幕分為五個區域：

    (2) Operation Menu
Execution button during program operation程式操作时的执行按钮
    (3) Program List
User-operable programs使用者可操作的程式
    (4) Data Manipulate
Data displayed when the program is executing程式执行时显示的数据
    (5) Login Information 
Database and local computer information displayed after user login
使用者登入后显示的数据库及本机电脑信息

Above is the Menu Bar. Click Exit to exit the window, or click X in the upper right corner of the window to close the window.
On the left is a list of programs. Only programs that can be executed by the logged-in user will appear.
The program window is on the right. Click the program list on the left to display the program window.
上方是Menu Bar，点选Exit可退出视窗，或点视窗右上角X可关闭视窗。
左边是程式清单，只会出现登入用户可执行的程式。
右边是程式视窗，点选左边的程式清单就会出现程式视窗。

The system must be configured before use系统在使用先必须先进行系统设定
    4. System Setting系统设定
Before the system starts to use, the system administrator must first establish the system user, the system program, and setup the right to use the program.
Setting the executable programs and operations performed by different users will restrict users' operations.
Users with the same role can create the users groups to facilitate the setting of program usage rights; but they can still individually set user program usage rights.
系统在开始使用前，系统管理员须先建立系统使用者、系统程式及设定程式的使用权。
设定不同的使用者可执行的程式及执行的操作，将会限制使用者的操作。
相同角色的使用者，可以创建、设定使用群组，方便操作程式使用权的设定；但仍然可以个别设定使用者程式使用权限。
